# Make Skill Connect Live

## Local Live Website

Run this command inside the project folder:

```powershell
npm.cmd start
```

Then open:

```text
http://127.0.0.1:5173/
```

Keep the terminal open. If you close it, the local website will stop.

## Online Live Website

To publish this project on the internet, use one of these options:

1. GitHub Pages
2. Netlify
3. Vercel

This project is a static website, so it can be hosted for free. Upload these files:

- `index.html`
- `styles.css`
- `app.js`
- `README.md`
- `PROJECT_REPORT.md`

For GitHub Pages, create a GitHub repository, upload the files, then enable Pages from repository settings.

For Netlify or Vercel, drag and drop the project folder or connect a GitHub repository.
