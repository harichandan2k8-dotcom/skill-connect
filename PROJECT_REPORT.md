# Skill Connect Project Report

## Title

Skill Connect: Find the Right Mentor

## Problem Statement

Students often need guidance for skills, projects, internships, and interviews, but they may not know which mentor is suitable for their goal. Manual mentor searching is time-consuming and can lead to poor guidance matching.

## Objective

The objective of Skill Connect is to provide a simple web application where students can discover mentors by skill, send mentorship requests, schedule sessions, communicate with mentors, and track learning progress.

## Proposed System

Skill Connect provides a common platform for students and mentors. Students can search mentors based on skills such as web development, Python, UI design, and project guidance. Mentors can accept or reject requests and help students through scheduled sessions and chat.

## Modules

1. Authentication Module
2. Mentor Directory Module
3. Skill Search Module
4. Mentorship Request Module
5. Session Scheduling Module
6. Progress Tracking Module
7. Chat Module

## Software Requirements

- Modern web browser
- Code editor such as Visual Studio Code
- Node.js only for optional local server

## Hardware Requirements

- Laptop or desktop computer
- Minimum 4 GB RAM
- Internet connection for background image loading

## Technologies Used

- HTML for structure
- CSS for design and responsive layout
- JavaScript for logic and interactivity
- localStorage for browser-based account and project data

## Advantages

- Easy mentor discovery
- Skill-based searching
- Simple request tracking
- Centralized session and chat system
- Beginner-friendly implementation

## Future Scope

- Database and backend login
- AI-based mentor recommendation
- Online video meetings
- Ratings and reviews
- Certificate and portfolio upload

## Conclusion

Skill Connect is a practical final year project that solves the problem of finding suitable mentors. It shows important application concepts such as login, role-based views, searching, request handling, scheduling, chat, and progress tracking.
